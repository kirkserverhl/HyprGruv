HyprGruv 🚀

Hyprland on Arch Linux with Gruvbox theming.
Developed by Kirk Bass

Prerequisites

  • Arch Linux ISO on a Ventoy USB (or other boot method)
  • Internet access during installation

Step 1–2: Install Arch Linux

Boot the ISO and run the guided installer:

archinstall

Suggested options:

SettingRecommendation
Diskbtrfs or ext4, compression on
Swapenabled
Bootloadergrub
ProfileDesktop → Hyprland, polkit
AudioPipeWire
NetworkNetworkManager
Extra packagesgit (firefox installed by Hyprgruv)

Reboot when finished. Remove the USB.

Step 3: Run Hyprgruv

At SDDM choose Hyprland (not uwsm-managed). Open a terminal (Win+Q during install; Win+Enter after).

sudo pacman -S git
git clone https://github.com/kirkserverhl/hyprgruv.git ~/.hyprgruv
cd ~/.hyprgruv
./install.sh

Install pipeline

On graphical sessions the full wizard runs in one pass before reboot. On TTY-only installs, reboot may happen first.

StepModule / scriptPurpose
100-preflight.shArch prep, keyring, multilib
201-packages.shyay, optional Chaotic-AUR, core stack + manifest lists
302-stow.shStow home/ configs into $HOME
4default_wp.shOpening wallpaper + matugen palette
5post_reboot_setup.shSetup wizard (03–05)
6Final syncyay -Syu or pacman -Syu
7RebootSkipped when a graphical session is detected

Setup wizard

Re-run anytime:

FORCE=1 bash ~/.hyprgruv/lib/scripts/post_reboot_setup.sh

Monitor layout is configured in Hyprland (conf/monitors.lua, save-monitor-layout.sh) — not part of the installer.

Environment variables

VariableEffect

SKIP_PACKAGES=1Skip package install

Tips

  • Move windows: Win + Left Mouse
  • Close during install: Win + C → after install: Win + Q
  • Keybind list: Win + K or type keybinds

Theming

Wallpaper-driven colors via matugen. See THEMING.md for Firefox setup, manual palette pickers, and troubleshooting.

Firefox one-time: enable toolkit.legacyUserProfileCustomizations.stylesheets in about:config, fix your profile path in ~/.config/matugen/config.toml, then change wallpaper once.

Post-installation

Log out and select Hyprland at SDDM. If anything was skipped:

FORCE=1 bash ~/.hyprgruv/lib/scripts/post_reboot_setup.sh

Legacy paths like ~/.dotfiles/install.sh are no longer used.
