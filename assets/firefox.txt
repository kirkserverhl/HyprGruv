# Firefox theming (HyprGruv / matugen)
#
# See assets/README/THEMING.md for the full guide.
#
# 1. about:config → toolkit.legacyUserProfileCustomizations.stylesheets = true
# 2. about:profiles → note your profile folder name
# 3. Update profile paths in ~/.config/matugen/config.toml (firefox templates)
# 4. Change wallpaper or run: matugen image ~/Pictures/Wallpapers/<image>
# 5. Restart Firefox
# 6. Uninstall Dark Reader and pywalfox extensions if still present
