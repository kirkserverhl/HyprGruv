# Legacy Rofi assets

Archived rofi templates and presets superseded by `~/.config/rofi/shared/`.

**Active rofi layout (do not move):**
- `~/.config/rofi/shared/` — `standard.rasi`, `fuzzel-chrome.rasi`
- `~/.config/rofi/config-*.rasi` — purpose-specific menus (compact, settings, calc, etc.)
- `~/.config/rofi/colors.rasi` — matugen output (or colorscheme copy via `apply-theme.sh`)
- `~/.config/settings/rofi-font.rasi`, `rofi-border.rasi`

**Archived here:**
- `launchers/` — old type-1…type-7 launcher style packs
- `themes/` — HyprGruv.rasi, wallSelect.rasi
- `share-themes/` — minimal/grid/wallpapers standalone themes
- `share-colors-custom/` — preset palette `.rasi` files for colorschemes
- `config.rasi`, `config-wallpaper.rasi`, `config-power.rasi`, `config-hyprshade.rasi`, etc.
