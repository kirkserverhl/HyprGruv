tmux learning cheat sheet (oh-my-tmux)
══════════════════════════════════════

Prefix = Ctrl-b   (stock tmux — safe when SSH'd without your config)

Sessions
  Ctrl-b d              detach (session keeps running)
  Ctrl-b s              session list (pick with arrows)
  Ctrl-b $              rename session
  Ctrl-b Ctrl-c         new session (oh-my-tmux)
  Ctrl-b Ctrl-f         find/switch session (oh-my-tmux)
  Ctrl-b BTab           last session (oh-my-tmux)

Windows (tabs)
  Ctrl-b c              new window
  Ctrl-b ,              rename window
  Ctrl-b w              window list
  Ctrl-b n / p          next / previous window
  Ctrl-b Ctrl-h / Ctrl-l   previous / next window (oh-my-tmux)
  Ctrl-b &              kill window (confirm)

Panes (splits)
  Ctrl-b -              split horizontal (top / bottom)
  Ctrl-b /              split vertical (left / right)
  Ctrl + arrows         move between panes (no prefix)
  Ctrl-b h j k l        move between panes (vim-style)
  Ctrl-b H J K L        resize pane (Shift + h/j/k/l)
  Ctrl-b Shift+arrows   resize pane (same as H J K L)
  Ctrl-b x              kill pane (confirm)
  Ctrl-b z              zoom pane (toggle)
  Ctrl-b { }            swap pane with previous / next
  Ctrl-b q              show pane numbers (then press number)
  Ctrl-b o              cycle panes
  Ctrl-b ;              last pane

Copy / scroll
  Ctrl-b [              copy mode (scroll + select)
  Enter                 copy mode (oh-my-tmux)
  q                     quit copy mode
  v                     start selection (vi mode)
  y                     copy selection to tmux buffer + system clipboard

Dev workspace (3-pane template)
  Alt+Return            new dev-N session in a terminal window
  Ctrl-b D              new dev-N session (from inside tmux)
  Ctrl-b s              session picker — switch between dev-1, dev-2, …
  Ctrl-b R              reset current dev session to the default layout
  dev-workspace --reset rebuild a named session, e.g. dev-workspace --reset dev-2

Config / help
  Ctrl-b ?              print this cheat sheet again (scroll up to read it)
  Ctrl-b r              reload tmux config (oh-my-tmux)
  Ctrl-b m              toggle mouse on/off

When you're comfortable, turn off auto-display:
  tmux set -g @learning_cheatsheet off
  (add that line to ~/.config/tmux/tmux.conf.local to make it permanent)
