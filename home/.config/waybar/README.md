As a token of gratitude, here are a couple of Waybar themes I've created.

## Steps to install

1. Back up your existing config and move this waybar folder into `~/.config/`

2. Bind Mod + R or your preferred keybind to execute `~/.config/waybar/scripts/launch.sh`

3. To change the theme, go into `launch.sh` and change the theme at the end of `waybar_config_dir` to one of the themes listed in the comment above.

Enjoy!
